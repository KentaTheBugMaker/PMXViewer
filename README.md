# PMXLoader
A simple PMX loader written in Rust
# What can this crate do
  Parse PMX 2.0/2.1 header
  Parse PMX 2.0/2.1 Model Info
    Name
    English Name
    Comment
    English Comment
  Parse Vertices Information
  Parse Material Information
